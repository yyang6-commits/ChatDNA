# ChatDNA Static Website

This version of website is for manuscript review purpose only.

## Quick Start

### For macOS/Linux:
```bash
./start.sh
```

### For Windows:
```bash
start.bat
```

The website will be available at: **http://localhost:5677**

## Manual Start

If you prefer to start manually:

```bash
python3 -m http.server 5677
```

## Stop Server

Press `Ctrl + C` in the terminal to stop the server.

## Requirements

- Python 3.x installed
- No additional dependencies required
